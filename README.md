# LINE-today-mockup
LINE TODAY website look-a-like

## Techs Used
- React
- Webpack
- Redux (Redux-saga)
- Swiper
- Global styles

## How to Run
1. Go to root directory LINE-today-mockup/
2. Install requirement
    ```sh
    npm install
    ```
    or
    ```sh
    yarn install
    ```
3. Start the program
    ```sh
    npm start
    ```
    or
    ```sh
    yarn start
    ```
4. Open localhost:8080/
